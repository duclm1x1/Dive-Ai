# Design Intelligence (UI/UX)

## Mục tiêu

Khi Agent làm UI:

- Bám theo 1 **style direction** rõ (ví dụ: Minimal, Neo-Brutalism, Glassmorphism, ...).
- Dùng design tokens (spacing, radius, type scale).
- Kiểm tra accessibility (contrast, focus states, keyboard nav).

## Tích hợp UI-UX-Pro-Max

Antigravity kit có thể tích hợp thêm UI-UX-Pro-Max như một pack styles riêng (see repo UI UX Pro Max). Khi có pack đó:
- `/ui-ux-pro-max` ... (nếu đã cài).
- Vibe v11 sử dụng style matrix để chọn nhanh direction.
