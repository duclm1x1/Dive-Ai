## üöÄ Quick Start

### 1. Initialize v19.6
```bash
cd dive-coder-v19.6
python3 -m dive_coder init
```

### 2. Use Hybrid Prompts
```python
from src.hybrid_prompt_system import HybridPromptSystem

system = HybridPromptSystem("My Project")

# Phase 1: VIBE
system.phase_1_vibe("Feature Name", "user action")

# Phase 2: VALIDATION
system.phase_2_validation(["feedback 1", "feedback 2"])

# Phase 3: PRO
system.phase_3_pro(["requirement 1", "requirement 2"])

# Phase 4: HYBRID
system.phase_4_hybrid(iterations=3)

# Get report
report = system.generate_report()
```

### 3. Generate Prompts
```bash
python3 tools/prompt_generator.py --feature "Social Feed" --phase vibe
```
