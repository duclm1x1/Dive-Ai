## ğŸš€ Quick Start

### Prerequisites
- Python 3.11+
- Node.js 22+ (for Dive Context)
- Git
- 4GB+ RAM

### Installation

```bash
# 1. Clone repository
git clone https://github.com/duclm1x1/Dive-Ai.git
cd Dive-Ai

# 2. Setup API keys (automated)
python3 setup_api_keys.py

# 3. Install Python dependencies
pip install -r requirements.txt

# 4. Install Dive Context (Node.js)
cd dive-context
pnpm install
pnpm build
cd ..

# 5. Install Monitor UI (optional)
cd ui/dive-monitor
npm install
cd ../..

# 6. Run startup checklist
python3 dive_ai_startup.py
```

### First Run

```bash
# Check system status
python3 dive_ai_startup.py

# Test LLM providers
python3 first_run_llm_test.py

# Track integration (self-aware memory)
python3 track_integration.py

# View changelog
cat CHANGELOG_V21.0.md
```
